# Membuktikan-Teori-Sampling-Kuantisasi-dan-Rekonstruksi-DAC-berbasis-MATLAB-pada-Audio-Smartphone
Simulasi MATLAB untuk membuktikan teori sampling Nyquist, kuantisasi, dan rekonstruksi DAC pada sistem audio smartphone. Proyek ini memvalidasi proses konversi sinyal digital ke analog agar dapat direproduksi melalui speaker tanpa distorsi aliasing dengan frekuensi sampling 44,1 kHz.
